The purpose of Gamify Inc. is to design and develop custom video games for instructional use.

Purpose of the website

The website forgamify.com is the main platform where the targeted audience, mostly students and instructors, can download video games and also communicate with and learn about the team of Gamify Inc.*

*Note that the website does not provide direct access to the video games developed by Gamify Inc. The user will be able to download the files necessary to download a video game launcher, which the students will have to input a key code that will enable the user to download the files for the desired game and course materials. Key codes will be purchased through student bookstores and other merchants that Gamify collaborates with.

To learn more about the downloadable game launchers, please refer to the Gamify Software Distribution Overview document.

Design Principles

Goal of website

To provide a platform that can:

  1. Provide a way to communicate with the team of Gamify Inc.
  2. Provide a means to learn more about Gamify Inc. and its team
  3. Provide a way for students and instructors to download files necessary for the desired game launcher
